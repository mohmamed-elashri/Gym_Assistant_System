# Contributing

Thank you for your interest in contributing to Gym Assistant System!

- Fork the repository and create a feature branch: `git checkout -b feature/your-feature`
- Write tests for new features or bug fixes.
- Run linters and formatters: `black`, `ruff`, `mypy` as configured in `pyproject.toml`.
- Commit with conventional commit messages (e.g., `feat:`, `fix:`, `chore:`).
- Push your branch and open a Pull Request describing the changes.

Maintainers will review and request changes if needed. For larger changes, open an issue first to discuss design and scope.
