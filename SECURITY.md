# Security Policy

If you discover a security vulnerability, please:

1. Do NOT open a public issue.
2. Contact the maintainers at `security@yourdomain.example` or open a private report via GitHub's security advisory.
3. Include reproducible steps and any relevant logs.

We will acknowledge the report and coordinate a fix and disclosure timeline.

For local secrets: remove any API keys from code and rotate them immediately if they were exposed.
